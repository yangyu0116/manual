类概述

         提供网络流量统计的类。这些统计包括通过所有网络接口、mobile接口和UID网络接口的字节发送和接收，网络数据包的发送和接收。

         这些统计可能不适用于所有平台。如果本设备不支持统计，UNSUPPORTED将被返回。

 

常量

public static final int UNSUPPORTED

返回值表示该设备不支持统计。

常量值: -1 (0xffffffff)
