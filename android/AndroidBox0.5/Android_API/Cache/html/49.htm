public static int getContantForFocusDirection(int direction)

针对不同声音方向，获得相应的发声常数。

参数

                            direction  参数取值为以下常量之一：FOCUS_UP, FOCUS_DOWN, FOCUS_LEFT, FOCUS_RIGHT, FOCUS_FORWARD , FOCUS_BACKWARD.

                   返回值

                            合适的发声常数
